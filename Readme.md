# 🌤 Smart Weather App

A modern and responsive weather application developed using **HTML, CSS, and JavaScript**. This project provides weather information, humidity, wind speed, temperature, date, and time with a beautiful user interface and dark mode support.

---

## 📌 Project Overview

Smart Weather App is designed to provide users with weather details in an attractive and user-friendly interface. The application demonstrates the use of modern front-end technologies and responsive web design principles.

---

## ✨ Features

* 🌡 Temperature Display
* ☁ Weather Condition Information
* 💧 Humidity Details
* 🌬 Wind Speed Information
* 🌡 Feels Like Temperature
* 🕒 Live Date and Time
* 🌙 Dark Mode Toggle
* 📱 Fully Responsive Design
* 🎨 Modern Glassmorphism UI
* 📅 5-Day Weather Forecast Section
* 👨‍💻 Developer Information Section
* ⚡ Smooth Transitions and Hover Effects

---

## 🛠 Technologies Used
